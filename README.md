Projeto Classificação Campeonato
Este sistema processa um arquivo de texto contendo resultados de partidas de futebol, calcula pontuação e saldo de gols, e exibe uma tabela de classificação ordenada.

Como configurar:
Certifique-se de que os arquivos 'Main.java' e 'jogos.txt' estejam na mesma pasta (raiz) do seu projeto.
O arquivo 'jogos.txt' deve seguir o formato: rodada,timeA,timeB,resultado (Ex: 1,Flamengo,Palmeiras,2x1).

Estrutura:
Main.java: Contém a lógica principal de leitura, processamento e exibição, além da classe auxiliar 'Time'.
jogos.txt: Base de dados com os resultados.
